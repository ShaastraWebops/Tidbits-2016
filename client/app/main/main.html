Event Format -

  • Tidbits is an individual event
  • The registration is online. Anyone with an email id can register on the website and participate in the event
  • Questions will be updated on a daily basis throughout the course of the event
  • There are no prelims, no finals. Got the right answer? Submit the answer and increase your points tally
  • The questions will be wide ranging. Most of the questions would be puzzles, ciphers
  • In addition, there will be a set of questions now and then whose answers contain concealed hints for solving subsequent questions
  • Follow the instructions while submitting answers

Instructions -

  • Every question has a unique answer and the answers are not case sensitive
  • There is no limit on the number of unsuccessful attempts to a question
  • If the solution to a question is a person or a place, the format of the answer would be the title of the corresponding Wikipedia article, unless stated otherwise. Eg : If the answer is Sachin Tendulkar, only "Sachin Tendulkar" (not case sensitive) would be considered as the correct answer. Other submissions like Sachin or Tendulkar or SachinTendulkar would be considered incorrect
  • If the solution is a number, the correct answer is the number in decimal system unless stated otherwise. Eg : If the answer is 6, only "6" would be considered as the correct answer. Other submissions like six would be considered incorrect
  • Leaderboard would be based on the number of correct answers submitted
  • Contact us for any other queries

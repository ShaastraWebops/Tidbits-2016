Instructions -

  • Every question has a unique answer and the answers are not case sensitive.
  • Eg : If the answer is Abcd Xyz, only "Abcd Xyz" (not case sensitive) would be considered as the correct answer. Other submissions like Abcd or Xyz or Abcdxyz would be considered incorrect.
  • There is no limit on the number of unsuccessful attempts to a question.
  • If the solution is a number, the correct answer is the number in decimal system unless stated otherwise. Eg : If the answer is 6, only "6" would be considered as the correct answer. Other submissions like "six" would be considered incorrect.
  • Leaderboard would be based on the number of correct answers submitted and the time of submission.
  • If you have solved all the questions, please wait for a few hours for the new level questions to be added.
  • Contact us for any other queries.
